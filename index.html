HTMl

È uma linguagem de Marcação de Hipertexto baseada em tags. As tags vão definir ao navegador a Marcaçãode conteúdo, cada tag tem funcionalidades especificas ex: < nome_tag > conteúdo < nome_tag >.

TAGS

Já as Tags de Heading são utilizados para difinir o titulo e possuem seis níveis ex: < h1 > titulo </h1 >. Para utilização de paragrafos utilizamos as Tags de Paragrafos ex:< p > conteudo < p >, a quebra pode ser feita com < br > ao final do texto.Para formatar tamanho da fonte utlizamos < big > conteudo < /b > ou < font size="tamanho da fonte" > conteudo < /font >, e para alterar a cor < font color="cor desejada" > conteudo < /font >. Para destacar um texto é possivel utilizar:

  • Negrito
  • italico
  • ânfase
  • Novo

Negrito < strong > conteudo < /strong > e para deixar um texto em negrito é < b > conteudo < /b >, para deixar um texto em itálico utlizamos < i > texto < /i >, já para dar ênfase é utilizado < em > texto < /em >
